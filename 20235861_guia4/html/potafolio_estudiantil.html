Desarrollo web

Desarrollo web es un curso diseñado para aprender todo tipo de temas esenciales en el desarrollo web, especialmente HTML, CSS, y Java, ademas, aprendemos a crear sitios responsivos y esteticos.

Guías de Ejercicios

  • Guía 1: Introducción y conceptos básicos de HTML y CSS

    Descripción: El obejtivo de esta guia es aprender a identificar la estructura basica de un documento web, aplicando reglas de estilo CSS a los elemento HTML.

    Ver Guía 1

  • Guía 2: Selectores CSS avanzados y modelo de caja

    Descripción: Con esta guia se espera que aprenda a utilizar y combinar los selectores avanzados de CSS, aprender sobre la importancia de las pseudo-clases y pseudo-elementos de CSS, ademas, se espera crear páginas web utilizando el Modelo de Cajas, identificando las respectivas partes.

    Ver Guía 2

  • Guía 3: Posicionamiento, visualización y maquetación de sitios web

    Descripción: Con esta guia aprenderemos a utilizar los diferentes tipos de posicionamiento CSS y diferenciarlos apropiadamente, tambien utilizaremos las propiedades CSS relacionadas con la visualización de elementos y maquetaremos sitios web con diseños fijos y fluidos.

    Ver Guía 3

  • Guía 4: Formularios e Introducción al Responsive Design

    Descripción: Esta guía abarca temas como la creación de formularios HTML5, el uso adecuado de la meta-etiqueta viewport, la aplicación de media queries para estilos según el ancho del navegador, y la construcción de sitios web responsivos.

    Ver Guía 4

  • Guía 5: Interactuando con JavaScript

    Descripción: Esta guía seremos capaz de generar código HTML desde secuencias de comando de JavaScript.

    Ver Guía 5

  • Guía 6: Estructuras de control y funciones en JS

    Descripción: esta guia nos permitira adquirir dominio en la construcción de funciones con JavaScript ,ademas, de hacer uso de parámetros o argumentos en las funciones que realiza.

    Ver Guía 6

FODA sobre el Desempeño

  • Fortalezas: Durante mis estudios de bachillerato, era obligatorio graduarse como técnica programadora en mi colegio. Lo primero que me enseñaron fue HTML y CSS; sin embargo, nunca profundizamos en la creación de sitios responsivos o formularios, ya que nos quedamos en lo básico. No obstante, gracias a estas clases y guías, he mejorado en la creación de estructuras HTML legibles y en su estilización con CSS.
  • Oportunidades: Aunque no estoy completamente segura del área a la que me quiero dedicar, estoy 100% convencida de que el aprendizaje del diseño responsivo, así como de HTML y CSS en general, será sumamente útil para mi futuro profesional, abriéndome las puertas a muchos proyectos.
  • Debilidades: Siendo el diseño responsivo algo nuevo para mí, admito que se me dificulta un poco; sin embargo, sé que con práctica se me hará más fácil con el tiempo.
  • Amenazas: Personalmente, siento que el tiempo fue uno de mis mayores enemigos con estas guías, ya que a menudo me llevaba más de dos horas terminarlas.
